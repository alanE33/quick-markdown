# 对角互补

## 基础模型

![image-20231207210905662](C:\Users\lenovo\AppData\Roaming\Typora\typora-user-images\image-20231207210905662.png)   

## 识别与证明

> 1. 任意四边形中有一组对角互补
>
> 2. 这组对角顶点连线平分其中一个角
>
> 3. 这组对角之一的角两边相等
>
>    （满足12或13即可）

> - 利用角平分线的性质作双垂线构造全等三角形；
>
> - 通过旋转以构造等腰三角形。
